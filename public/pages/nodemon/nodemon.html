Nodemon

What is Nodemon?

Nodemon is a tool that monitors your project files and automatically restarts your server when it detects changes.


How to install Nodemon?

Nodemon works best when installed globally using the -g argument

npm install -g nodemon

Usage

In development, run you project with `nodemon app.js`

nodemon app.js
[nodemon] 2.0.15
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,json
 [nodemon] starting `node app.js`

The server will now restart automatically when detecting changes in your project. If you need to force a restart for any reason you can type rsâ†µ in the terminal


Settings

Nodemon automatically looks for a start script in package.json. If you have specified one, you can don't have to include the entry point argument when starting a server with Nodemon

"scripts": {
  "start": "node app.js"
}, nodemon
[...]
[nodemon] starting `node app.js`

By default Nodemon only monitors files with certain file extensions. It is possible to change these defaults by running the server with the -e or --ext argument.

nodemon -e js,json,css,html
[...]
[nodemon] watching extensions: js,json,css,html
 [nodemon] starting `node app.js`

You can also specify settings inside a designated config file, nodemon.json

{
  "ext": "js,json,css,html",
  "...": "... other values",
}

Or even inside package.json if you want to keep everything in one place

"nodemonConfig": {
  "ext": "js,json,css,html"
  "...": "... other values",
},

For further documentation see https://github.com/remy/nodemon#readme
