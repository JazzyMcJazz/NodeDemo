NPM

What is NPM?

NPM (originally short for Node Package Manager) is a package manager for JavaScript and the world's largest software registry. NPM comes with a CLI which grants the user access to many useful features (see below).


Usage

Users can use NPM to install tools globally in their development environment to help assist in development. Locally (per independent project) NPM is used to install and manage packages (dependencies).


Common Commands

npm init

Starts a wizard that initializes a new project with the current directory as root. At the conclusion of the wizard a package.json file is created.


npm install

Installs or updates dependencies found in package.json


npm install <package_name>

Adds the package to the list of dependencies found in package.json and installs it


package.json

An example of a package.json file:

{
  "name": "project_name",
  "version": "1.0.0",
  "description": "describes the project",
  "main": "app.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node app.js"
  },
  "repository": {
    "type": "git",
    "url": "git+https://github.com/SomeUser/SomeRepo.git"
  },
  "keywords": [
    "npm",
    "express",
  ],
  "author": "John Smith",
  "license": "MIT",
  "dependencies": {
    "express": "^4.17.3"
  }
}

Scripts

You can write scripts that can be called by NPM with a custom keyword by adding them under scripts in your package.json file.

Common examples are scripts for testing and executing your project:

"scripts": {
  "test": "echo \"Error: no test specified\" && exit 1",
  "start": "node app.js"
},

Finding packages
