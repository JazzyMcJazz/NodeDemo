Node.js

What is Node.js?

Node.js is a runtime environment that allows JavaScript to run outside the client (i.e. browser) much like many other programming language like Java, Python, etc.


One of the most common use cases for Node.js is running webservers serving REST APIs or even server side rendering.


How to install Node.js?

You can download and install Node.js from https://nodejs.org/


Running a Node.js application

Using the command below will execute a given JavaScript file. In this case, the file is called app.js

node app.js
