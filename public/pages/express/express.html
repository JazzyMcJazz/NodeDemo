Express

What is Express?

Express is a web application framework for Node.js. It supplies out of the box server functionality such as HTTP controller methods (GET, POST, PUT, DELETE, etc.) and many options for server behavior


How to Install Express?

As with any NPM package, there are two ways you can install Express:


Option 1:
Declare express under dependencies in package.json

"dependencies": {
  "express": "^4.17.3"
}

and run

npm install

Option 2:
Simply run

npm install express

How to import express into your project?

Simply instantiate the two objects as below. (Note: the require keyword is how you import modules in Node.js)

const express = require( 'express');
const app = express();

Usage

You can use express to declare HTTP controller methods

const cars = [];

app.post('/cars', (req, res) => {
    cars.push(req.body);
    res.send(cars);
});

app.get('/cars', (req, res) => {
    res.send(cars);
});

You can configure Express to interpret the data contained in an incoming REST API calls as a specific data type, for example JSON. This would be necessary for the code in the previous example to work.

app.use(express.json());

You can configure a certain folder and its sub folders where certain assets are contained to be public so that a client can access them without an explicit pointer from the controller. Such assets could be html, css and JavaScript files.

app.use(express.static('public'));

You can configure Express to listen on a certain port. This is necessary in order to run a functioning web server. By convention this statement should be a the bottom of your project

app.listen(8080);

You can configure a router object in Express to define your endpoints. This is practical if you want to put your HTTP methods in a separate file:

// router.js

const router = require('express').Router();

router.get('/some-endpoint', (req, res) => res.send(...));

module.exports = router;
// app.js

const router = require('./router.js');

app.use(router);

Additional documentation can be found at https://expressjs.com/
